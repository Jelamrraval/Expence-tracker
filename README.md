# 💰 Expense Tracker

## Description

Expense Tracker is a web-based application developed using Python and Flask to help users manage their daily expenses efficiently. The application allows users to add, view, and delete expense records while automatically calculating total expenses. It provides a simple, responsive, and user-friendly interface for personal finance management.

## Features

* Add new expense records
* Delete existing expense records
* View expense history in a tabular format
* Automatic total expense calculation
* Date-wise expense tracking
* Expense categorization
* Description support
* Responsive user interface
* Flash notification system
* CSV-based data storage

## Technologies Used

### Backend

* Python
* Flask

### Frontend

* HTML5
* CSS3
* Bootstrap 5
* JavaScript

### Data Storage

* CSV File

## Flask Features Used

### Flash Messages

Flask Flash Messages provide instant feedback to users for actions such as:

* Expense added successfully
* Expense deleted successfully
* Validation error messages

### Routing

Flask routes handle user requests and connect frontend pages with backend logic.

### Template Rendering

HTML pages are dynamically rendered using Flask templates.

## Project Objective

The objective of this project is to provide an easy-to-use platform for recording and managing daily expenses, helping users maintain financial records in an organized manner.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Jelamrraval/Expence-tracker.git
```

2. Navigate to the project folder:

```bash
cd Expence-tracker
```

3. Install required packages:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```

5. Open your browser and visit:

```text
http://127.0.0.1:5000
```
