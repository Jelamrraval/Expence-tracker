# Expence-tracker

## Description
Expense Tracker is a web-based application developed to help users manage their daily income and expenses efficiently. The system allows users to record transactions, categorize expenses, monitor spending habits, and view financial summaries. It provides a simple and user-friendly interface for tracking personal finances and maintaining budget control.

## Features
- Add, edit, and delete income and expense records
- Categorize transactions
- View transaction history
- Track total income, expenses, and balance
- Search and filter transactions
- Responsive and user-friendly design
- Secure user authentication and authorization

## Technologies Used
- Python
- Django Framework
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- SQLite Database

## Project Objective
The main objective of this project is to provide an easy-to-use platform for managing personal finances, tracking expenses, and helping users make informed financial decisions through organized record keeping.

## Installation
1. Clone the repository.
2. Install required dependencies.
3. Run database migrations.
4. Start the Django development server.
5. Open the application in your web browser.
